<Info>
Ordered list functionality is part of the [Paragraph extension](/extensions/paragraph). There is no separate ordered list extension.
</Info>

## Commands

### `toggleOrderedList`

Toggle ordered list formatting on the current selection. Converts selected paragraphs to numbered list items or removes list formatting.

**Example:**

```javascript
editor.commands.toggleOrderedList()
```

### `restartNumbering`

Restart list numbering from the current position.

**Example:**

```javascript
editor.commands.restartNumbering()
```

Type `1. ` followed by a space to start a numbered list automatically.
