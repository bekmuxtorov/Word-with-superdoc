## Options

Configure the extension behavior:

<ParamField path="types" type="Array<string>" default="['textStyle','tableCell']">
  Node/mark types to add font size support to
</ParamField>

<ParamField path="defaults" type="FontSizeDefaults">
  Default size configuration
</ParamField>

## Attributes

Node attributes that can be set and retrieved:

<ParamField path="fontSize" type="FontSizeValue">
  Font size with unit
</ParamField>

## Commands

### `setFontSize`

Set font size

<Note>
Automatically clamps to min/max values
</Note>

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.setFontSize('14pt')
editor.commands.setFontSize('18px')
editor.commands.setFontSize(16)
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.setFontSize('14pt')
    editor.commands.setFontSize('18px')
    editor.commands.setFontSize(16)
  },
});
```
</CodeGroup>

**Parameters:**

<ParamField path="fontSize" type="FontSizeValue" required>
  Size to apply (with optional unit)
</ParamField>

### `unsetFontSize`

Remove font size

<Note>
Reverts to default document size
</Note>

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.unsetFontSize()
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.unsetFontSize()
  },
});
```
</CodeGroup>

## Types

### `FontSizeDefaults`

Font size configuration

<Expandable title="Properties">
<ResponseField name="value" type="number">
  Default font size value
</ResponseField>
<ResponseField name="unit" type="string">
  Default unit (pt, px, em, rem)
</ResponseField>
<ResponseField name="min" type="number">
  Minimum allowed size
</ResponseField>
<ResponseField name="max" type="number">
  Maximum allowed size
</ResponseField>
</Expandable>

### `FontSizeValue`

Size with optional unit (e.g., '12pt', '16px', 14)
