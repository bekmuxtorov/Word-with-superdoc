## Commands

### `insertLineBreak`

Insert a line break

<Note>
Creates a soft break within the same paragraph
</Note>

**Example:**

```javascript
editor.commands.insertLineBreak()
```

### `insertPageBreak`

Insert a page break

<Note>
Forces content to start on a new page when printed
</Note>

**Example:**

```javascript
editor.commands.insertPageBreak()
```
