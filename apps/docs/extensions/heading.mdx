## Options

Configure the extension behavior:

<ParamField path="levels" type="Array<number>" default="[1,2,3,4,5,6]">
  Supported heading levels
</ParamField>

## Attributes

Node attributes that can be set and retrieved:

<ParamField path="level" type="number" required>
  Heading level (1-6)
</ParamField>

## Commands

### `setHeading`

Set a heading with specified level

<Note>
Converts current block to heading
</Note>

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.setHeading({ level: 2 })
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.setHeading({ level: 2 })
  },
});
```
</CodeGroup>

**Parameters:**

<ParamField path="attributes" type="HeadingAttributes" required>
  Heading attributes including level
</ParamField>

### `toggleHeading`

Toggle between heading and paragraph

<Note>
Switches between heading and paragraph for the same level
</Note>

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.toggleHeading({ level: 1 })
editor.commands.toggleHeading({ level: 3 })
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.toggleHeading({ level: 1 })
    editor.commands.toggleHeading({ level: 3 })
  },
});
```
</CodeGroup>

**Parameters:**

<ParamField path="attributes" type="HeadingAttributes" required>
  Heading attributes including level
</ParamField>
