## Commands

### `setStrike`

Apply strikethrough formatting

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.setStrike()
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.setStrike();
  },
});
```
</CodeGroup>
