## Options

Configure the extension behavior:

<ParamField path="types" type="Array<string>" default="['textStyle']">
  Mark types to add color support to
</ParamField>

## Attributes

Node attributes that can be set and retrieved:

<ParamField path="color" type="ColorValue">
  Text color value
</ParamField>

## Commands

### `setColor`

Set text color

<Note>
Preserves other text styling attributes
</Note>

**Example:**

<CodeGroup>
```javascript Usage
// Set to red using hex
editor.commands.setColor('#ff0000')
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    // Set to red using hex
    editor.commands.setColor('#ff0000');
  },
});
```
</CodeGroup>

**Parameters:**

<ParamField path="color" type="ColorValue" required>
  Color value to apply
</ParamField>

### `unsetColor`

Remove text color

<Note>
Removes color while preserving other text styles
</Note>

**Example:**

<CodeGroup>
```javascript Usage
editor.commands.unsetColor()
```

```javascript Full Example
import { SuperDoc } from 'superdoc';
import 'superdoc/style.css';

const superdoc = new SuperDoc({
  selector: '#editor',
  document: yourFile,
  onReady: (superdoc) => {
    const editor = superdoc.activeEditor;
    editor.commands.unsetColor();
  },
});
```
</CodeGroup>

## Types

### `ColorValue`

Accepts hex colors (#ff0000), rgb(255,0,0), or named colors (red)
