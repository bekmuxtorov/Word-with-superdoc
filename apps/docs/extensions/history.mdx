## Options

Configure the extension behavior:

<ParamField path="depth" type="number" default="100">
  Maximum undo/redo steps to remember
</ParamField>

<ParamField path="newGroupDelay" type="number" default="500">
  Milliseconds to wait before starting a new history group
</ParamField>

## Commands

### `undo`

Undo the last action

<Note>
Groups changes within the newGroupDelay window
</Note>

**Example:**

```javascript
editor.commands.undo()
```

### `redo`

Redo the last undone action

<Note>
Only available after an undo action
</Note>

**Example:**

```javascript
editor.commands.redo()
```

## Keyboard shortcuts

| Command | Shortcut | Description |
|---------|----------|-------------|
| undo() | `⌘/Ctrl-z` | Undo last action |
| redo() | `⌘/Ctrl-Shift-z` | Redo last action |
| redo() | `⌘/Ctrl-y` | Redo last action (alternative) |
